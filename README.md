# 🛒 FreshBasket - AI-Powered Grocery Ordering App

A full-stack grocery ordering platform inspired by Blinkit, featuring AI-powered recipe suggestions, secure payments, and complete order management.

## ✨ Features

- 🔐 **Authentication** - Email/Password, Google OAuth, Phone OTP with JWT
- 🛍️ **Product Management** - 30+ products with categories, search, and variants
- 🤖 **AI Recipe Tool** - Enter any dish, AI generates ingredients with quantities
- 🛒 **Smart Cart** - Persistent cart (localStorage + database sync)
- 💳 **Payments** - Razorpay integration with test mode
- 📦 **Order Management** - Order history, invoice generation, address saving
- 👑 **Admin Panel** - Add/Edit/Delete products (role-based access)
- 📱 **Responsive UI** - Blinkit-inspired design with Tailwind CSS

## 🛠️ Tech Stack

### Backend
- Node.js + Express.js
- MongoDB with Mongoose
- JWT Authentication
- Razorpay API
- GROQ API (AI Recipe)

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Context API
- React Router DOM
